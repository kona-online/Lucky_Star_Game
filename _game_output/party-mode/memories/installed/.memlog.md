- (decision) 장르 방향 확정: 캐릭터별 미니게임 모음(메타게임) 구조 - 코나타처럼 오타쿠 미니게임들을 깨부수며 각 캐릭터 매력을 느끼게 함
- (decision) 배포 모델 확정: 비상업 동인 배포 전제
- (direction) 정해진 마감 없음 - 발산(이상적인 모습)을 먼저 다 펼치고 나중에 추리는 방식으로 진행
- (decision) 스코프 확정: 주인공 4인방(코나타/카가미/츠카사/미유키) 중심으로 우선 제작, 추가 요소는 개발하며 결정
- (decision) 코나타 미니게임 확정: 쿠로이 나나코 선생님 수업 중 졸다가 선생님이 뒤돌아볼 때 깨어있어야 하는 자원관리+반사신경 게임 (유저 아이디어)
- (decision) 카가미 미니게임 확정: 짧은 스테이지형 QTE 츳코미 게임, '오늘의 헛소리' 테마로 매번 다른 상대(코나타/츠카사/미유키)와 진행
- (decision) 츠카사 미니게임 확정: 카루타식 그림 맞추기 게임 (차분한 템포의 휴식 챕터)
- (outcome) 주인공 4인방 미니게임 장르 전부 확정: 미유키=퀴즈, 코나타=수업 중 졸음 자원관리+반사신경, 카가미=QTE 츳코미 스테이지형, 츠카사=카루타 그림 맞추기
- (decision) 전체 프레임 확정(유저 아이디어, 강한 호응): 2027년 20주년 설정 - 성인이 된 코나타가 사진앨범을 열면 네 장의 사진이 있고 그게 각 챕터(미니게임)가 되는 회상 구조. 기존 '코나타가 게임 만듦' 메타프레임 대체
- (outcome) 전체 구조 확정: (1)네 사진은 자유 순서로 플레이, 클리어 시 사진에 '클리어' UI/다른 버전 사진 노출 (2)4장 전부 클리어 시 엔딩 - 어른이 된 카가미/츠카사/미유키가 밖에서 코나타를 기다리고 부르면, 코나타가 '지금 갈게~' 하며 나가고, 네 명이 한 컷에 모이는데 어른이 아닌 고등학교 시절 모습으로 등장하는 연출
- (outcome) 세션 마무리: 전체 기획(장르/프레임/4챕터/엔딩) 확정 완료. 다음 단계로 개발 환경 선정과 각 미니게임 구체적 기획(조작, 화면, 디테일)이 남음 - 다음 세션에서 이어갈 주제
